# Sylithe-Verify
India's Carbon Credit Trading Scheme (CCTS) is live — but post-issuance monitoring remains a blind spot. Projects verified at onboarding are rarely re-checked. SEBI's BRSR Core now demands ongoing evidence. Sylithe Verify bridges this gap using satellite intelligence and AI-driven analysis.

Satellite Monitoring
Continuous LULC change detection using Sentinel-2 and canopy height models.

BRSR Core Compliance
Audit-ready scoring for SEBI's mandatory sustainability reporting framework.

Carbon Verification
Compare onboarding claims with real-time satellite observations.

Alert System
Automatic alerts for deforestation, fire, encroachment, and land use change.

This project was developed by Jasleen Kaur as part of her MBA research in Sustainable Finance. It demonstrates how satellite-based verification can bring transparency and accountability to India's nascent carbon credit market, ensuring that environmental claims are backed by real, measurable, and continuously monitored evidence.
